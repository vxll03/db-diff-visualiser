## 📖 About The Project

**VantageDB** bridges the gap between database evolution and documentation. By automatically analyzing Alembic migration files and the current PostgreSQL state, it generates up-to-date documentation and visualizes schema changes. It provides a transparent view of your architecture.

## ✨ Features

* 🔄 **Auto-Documentation:** Generates accurate schema docs directly from applied migrations.
* 📊 **Alembic Visualization:** Creates intuitive visual graphs of your migration history and branches.
* ⚡ **Seamless Integration:** Designed specifically for the `FastAPI`, `SQLAlchemy`, and `PostgreSQL` ecosystem.
